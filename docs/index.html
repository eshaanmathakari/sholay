02 / what it takes to run

Three macOS permissions, no sudo.

The agent needs eyes, hands, and the ability to talk to apps — three TCC grants on the host terminal. Nothing privileged: the demo flows install nothing and safe_shell() hard-refuses sudo.

Screen Recording
The agent's eyes — every step shoots the screen via screencapture -x.
Accessibility
The agent's hands — pyautogui synthesizes clicks, keystrokes, scroll, hotkeys.
Automation
Talk to apps — osascript activates apps and opens URLs; prompts per app.
No sudo
safe_shell() refuses sudo, rm, disk-erase, and anything outside a read-only allow-list.

Runtime: macOS + display · Python 3 · pip install -r requirements.txt  ·  Secret: ANTHROPIC_API_KEY in a gitignored .env  ·  Preflight: agent.py --diagnose probes the capture + input paths before spending tokens.
